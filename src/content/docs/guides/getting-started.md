本指南将帮助您快速上手并开始使用我们的文档系统。

## 安装

### 前置条件

在开始之前，请确保您的系统中已安装以下工具：

- Node.js (v16.12.0 或更高版本)
- npm 或 yarn 或 pnpm

### 创建新项目

使用以下命令创建一个新的 Astro 项目：

```bash
# 使用 npm
npm create astro@latest

# 使用 yarn
yarn create astro

# 使用 pnpm
pnpm create astro
```
